# react-native-emoji-picker

React Native Emoji Picker

## Installation

```sh
npm install react-native-emoji-picker
```

## Usage

```js
import { EmojiPickerView } from "react-native-emoji-picker";

// ...

<EmojiPickerView color="tomato" />
```
